Math Magic


In this project, you will become a mathemagician and write a small program that performs a mathematical magic trick! It will involve performing arithmetic operations on an integer that you choose.

Follow the instructions in the next video (https://youtu.be/Roz_okjnGNM)  and complete the Magic.java program.

When you finish, transform the program so that it uses the Scanner object to request an integer number (myNumber) by keyboard.
